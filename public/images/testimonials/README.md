## Image Specifications

- Format: JPG, PNG, or WebP
- Size: 400x400px minimum (square aspect ratio)
- File size: Keep under 500KB for optimal loading
- Professional headshots work best

## Fallback

If images are not available, the component will show initials with a gradient background as a fallback.
